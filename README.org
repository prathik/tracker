* Pomodoro Tracker
** Summary
Track the number of pomodoro sessions done
** Background
Currently I operate via pomodoro, running a total of 12 sessions of 25 minutes each. One of the key issues is a lack of fast way to track the number of pomodoro sessions that done.
** Goals
1. Track total pomodoro donerun should be visible
2. Have a quick and easy way to track the same
3. The Pomodoro sessions trend should be visible
** Non Goals
1. Tracking non pomodoro work
** Installation

#+begin_src sh :results verbatim :exports both
go get -u github.com/prathik/tracker
#+end_src

** Usage
*** Increment data for current session
#+begin_src sh :results verbatim :exports both
tracker increment
#+end_src
*** Show previous session data
#+begin_src sh :results verbatim :exports both
tracker show
#+end_src
*** Show a graph of joy and importance of activities
#+begin_src sh :results verbatim :exports both
tracker show graph
#+end_src
